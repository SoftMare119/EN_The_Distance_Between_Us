Order System Operation Procedures

Administrator Operations Guide - Please strictly follow these procedures for system operations

System Access Procedures

1. Login Permission Requirements

Only authorized administrators may access the order management system. Sharing login credentials or access links with any third party in any form is prohibited. Each administrator must use unique login credentials.

2. Device Security Requirements

System access must be conducted in a secure network environment. Using public WiFi or shared devices to access the system is prohibited. You must completely log out of your account before leaving your workstation.

3. Order Processing Procedures

All orders must be processed within the specified time:

  • Standard orders: Process within 48 hours
  • Special orders: Process within 24 hours
Important Notice: Please visit the product display to ensure all order information is up to date. Delete public records immediately after processing orders.
